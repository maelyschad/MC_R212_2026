# Rapport — TD1

## Points clés

### 1. Découper une interface en blocs logique 

L'exercice de découpage en bloc d'une interface permet de mieux comprendre la structure de fond du site, comment il est construit, et permet de visualiser la segmentation du code derrière. 

### 2. Structure d'une page html

Header / Hero / Main / Section / Footer 
On comprend également le rôle des balises sémantiques <header>, <div>, <nav>,... 
L'organisation d'une page html est importante pour un code compréhensible. 

### 3. Rôle de chaque langage de code: HTML, CSS et Javascript

HTLM -> contenu
CSS -> apparence 
Javascript -> intéractions

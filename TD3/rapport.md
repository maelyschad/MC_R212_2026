# Rapport — TD3

## Points clés

### 1. Expliquez le modèle événement → état → rendu avec un exemple de votre choix

L'évènement est l'action éffectuée qui déclenche la suite du code. L'état est la résultante de l'évènement, ce qui est mis en place en conséquence de l'action. Le rendu est le changement obtenu, ce qui a changé par rapport à l'état initial. 
Exemple: mettre un like sur une publication. 
évènement --> Clic sur le petit coeur
état --> +1 dans le compteur de likes
Rendu --> Le nombre de likes affiché a augmenté de 1

### 2. Expliquez en quoi classList.toggle est la passerelle entre JS et CSS

classList.toggle permet d'ajouter ou de retirer une classe instantanément, le CSS gère la classe, son apparance, et le JSS détermine à quel moment la classe apparaît ou disparaît.

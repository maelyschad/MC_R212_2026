# Rapport — TP3

## Points clés

### 1. Insérer du html dans le js

... .innerHTML = `` --> injecte du html dans les variables
document.createElement() --> créer un élément html

### 2. Filtrer

.filter() --> créer un tableau qu'avec les éléments qui passent le test
.includes() --> vérifier si un tableau contient une valeur

### 3. Fonctions

On peut créer une fonction function Nom() {}, puis la rappeler quand on veut dans le code, juste en tapant son nom --> Nom();

### 4. Stockage Json

Il est possible de stocker des données localement dans le navigateur, cela évite les suppressions lors du rechargement de la page.
localStorage.setItem(clé, valeur) --> stockage dans le navigateur
localStorage.getItem(clé) --> récupérer les données stockés 
JSON.stringify() --> convertir un objet/tableau JS en texte
JSON.parse() --> inversement, convertir un texte en objet/tableau JS

# Rapport — TD2

## Points clés

### 1. Fonction du CSS

Le css permet la stylisation d'une page web. C'est avec celui-ci que l'on gère la police, la taille des textes, la disposition des items, les couleurs... il ne change pas le fond, simplement la forme. 

### 2. Utilisation CSS

On peut dédié des commandes css à tout les blocs ou à chacun précisemment, ou à une classe particulière. header { ....}, .classe {...}, h2 {...}

### 3. Flex

Flex permet de gérer l'emplacement, honrizontalement et verticalement, des éléments choisis, ainsi que l'enroulement lors de la diminution de taille de l'écran. Ici utilisé de manière la plus simple pour la liste nav.
